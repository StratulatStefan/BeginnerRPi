Host Website

Una dintre cele mai frecvente intrebari care apare in randul dezvoltatorilor de pagini web, este referitoare la hosting. Unde anume sa gazduiesti un server web? Cei mai multi apeleaza la servicii platite de hosting. Pentru cei mai multi, aceasta este cea mai buna solutie, intrucat companiile de hosting ofera multe garantii, cum ar fi numar nelimitat de accesari fara erori, ceea ce ar fi mai greu de realizat printr-un hosting creat individual.

Cand vine vorba de platforme de hosting construite individual, multi dezvoltatori dau un pas inapoi, fiind constienti de costurile ridicate ale acestora. Cu toate acestea, in cazul in care dorim sa dezvoltam o pagina web care nu are foarte multe accesari, dar care sa ruleze constant fara erori, putem sa ne gandim la platforme precum Raspberry Pi, mai ales ca ultimele generatii sunt foarte puternice, avand multa putere de calcul si memorie.

Pentru a demonstra acest lucru, am realizat 2 servere :

  1. Un server realizat cu framework-ul Flask pentru Python, server care gazduieste aceasta pagina
  2. Un server realizat cu NodeJS, care va gazdui un site mai complex, ce necesita multe resurse
Click aici pentru a accesa site-ul

Site-ul este realizat pe platformat NodeJS si necesita foarte multe resurse

  • Comunicare cu server de baze de date de tip SQL, instalat pe RPi
  • Manipulare de fisiere de tip JSON
  • Foarte multe cereri de tip HTTP pentru resurse web (toate imaginile sunt preluate de pe internet)
  • Manipulare de cookie-uri pentru datele utilizatorilor
  • Conectare la server STMP.Gmail pentru transmiterea de email-uri
  • Numar mare de fisiere HTML, CSS si Javascript

Cu toate acestea, RPi gazduieste foarte bine acest server de NodeJS, oferind acces garantat in orice moment al zilei, unui numar mare de utilizatori.
